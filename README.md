WIP

## Usage

### Install

Clone and install dependencies:

```bash
git clone git@github.com:MartinTale/red-right-fist.git
cd red-right-fist
npm i
```

### Dev server

Start the dev server with hot reload:

```bash
npm run dev
```

Open your web browser to <http://localhost:5173/>

### Production build

Create a final production build:

```bash
npm run build
```
